\doxysubsection{Detailed Description}
Library for the \mbox{\hyperlink{class_d_s1307}{DS1307}} Real Time Clock. 

This is the library for controlling and reading the \mbox{\hyperlink{class_d_s1307}{DS1307}} RTC (Real Time Clock). It uses the library hwlib made by (c) Wouter van Ooijen (\href{mailto:wouter@voti.nl}{\texttt{ wouter@voti.\+nl}}) 2017. Without hwlib this library does not work. ~\newline
