Detailed Description

Library for the DS1307 Real Time Clock.

This is the library for controlling and reading the DS1307 RTC (Real Time Clock). It uses the library hwlib made by (c) Wouter van Ooijen (woute.nosp@m.r@vo.nosp@m.ti.nl) 2017. Without hwlib this library does not work.

Fuction to control the SQW/out pin

Controls the SQW/out pin. The parameter output_control controls the pin when square_wave is disabled. The parameter square_wave turns the oscilator output on or off. The rate_select parameter controls the rate. If rate_select is 0 then the rate is 1 Hz; if it is 1 then the rate is 4.096 kHz; 2 equals 8.192 kHz and 3 equals 32.768 kHz.
